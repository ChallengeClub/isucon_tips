## ユーザ切替  
```
$ sudo su - isucon
```

## インストール  
```
$ sudo apt install dstat
```

## 環境調査  
```
$ cat /etc/os-release                   # os確認  
$ cat /etc/passwd | grep -v nologin     # user確認  
$ sudo lsof -P -i | grep -v sshd        # process確認
$ service --status-all                  # service確認
```
