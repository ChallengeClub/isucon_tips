# NewRelic
New Relicのユーザー登録の際に、Userグループにしておく必要あり。
無料アカウントでは、代表以外の人はBasic Userとする。

# ホストにAgentインストール

AddData -> Guided Install -> Linux -> インストールコマンドが表示される。
EC2 インスタンス上で指示されたコマンドを実行すると、NewRelicのAgentがインストールされる。
